## Overview

The MavunoX Recommendation Engine uses OpenAI's GPT-3.5 Turbo API to generate custom, actionable farming recommendations for rural farmers. The engine analyzes environmental factors, compares them against crop-specific optimal ranges, and produces concise, farmer-friendly advice to improve crop yield.

## Architecture

### Model Integration
- **Model**: GPT-3.5 Turbo
- **API**: OpenAI Chat Completions
- **Response Length**: 55 words (strictly enforced)
- **Output Format**: Block of text, simplified for rural farmers

## Implementation

### Core Function

```python
from openai import OpenAI
from dotenv import load_dotenv
import os

load_dotenv()
api_key = os.getenv('OPENAI_API_KEY')

client = OpenAI(api_key=api_key)

def rec_gen(factor, min, max, value, unit, crop):
    """
    Generate farming recommendation using GPT-3.5 Turbo.
    
    Args:
        factor (str): Environmental factor (e.g., "Temperature", "pH")
        min (float): Minimum recommended value
        max (float): Maximum recommended value
        value (float): Current measured value
        unit (str): Unit of measurement (e.g., "Celsius", "mm")
        crop (str): Target crop name
    
    Returns:
        str: Recommendation text (55 words)
    """
    completion = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {
                "role": "system", 
                "content": "You are a recommender for rural farmers. only use block of text in 55 words strictly and make responses dynamic"
            },
            {
                "role": "user", 
                "content": f"Provide a recommendation to improve yield based on {factor}. The recommended {factor} is {min}-{max} {unit}, and the farm is currently at {value} {unit}. In just 55 words, assess if the current {factor} is suitable for optimal yield and add strategies to improve yield specific for the crop {crop}, recommendation should be block of text and simplified, approximate to 2 decimal places for all numbers in responds"
            }
        ]
    )
    return completion.choices[0].message.content
```

## Recommendation System

### Status Calculation

Before generating recommendations, MavunoX calculates whether current values are within acceptable ranges:

```python
def calc_status(value, value_range):
    """
    Check if value is within acceptable range.
    
    Args:
        value (float): Current measured value
        value_range (list): [min, max] acceptable range
    
    Returns:
        int: 1 if within range, 0 if outside
    """
    if value >= value_range[0] and value <= value_range[1]:
        status = 1
    else:
        status = 0
    return status
```

### Scale Calculation

Calculates how close the current value is to optimal using Gaussian function:

```python
import math

def calc_scale(actual_number, input_number, k):
    """
    Calculate proximity to optimal value (0-100 scale).
    
    Args:
        actual_number (float): Current value
        input_number (float): Optimal value
        k (float): Sensitivity parameter
    
    Returns:
        float: Score from 0-100 (100 = optimal)
    """
    absolute_difference = abs(actual_number - input_number)
    scale_value = 100 * math.exp(-k * absolute_difference**2)
    return scale_value
```

## Recommendation Characteristics

### Design Principles

1. **Brevity**: Strict 55-word limit for easy comprehension
2. **Simplicity**: Language tailored for rural farmers
3. **Actionable**: Specific strategies, not just analysis
4. **Crop-Specific**: Recommendations tailored to target crop
5. **Dynamic**: GPT generates unique responses based on input
6. **Precision**: Numbers rounded to 2 decimal places

### Response Format

Each recommendation includes:
- Current value assessment
- Comparison to optimal range
- Specific action items
- Expected benefits
- Crop-specific considerations

## Best Practices

### API Usage

- Store API key securely in environment variables
- Implement rate limiting to avoid OpenAI quota issues
- Cache recommendations for identical inputs
- Handle API errors gracefully
- Monitor API usage and costs

### Prompt Engineering

- System prompt enforces 55-word limit strictly
- User prompt provides all context in single message
- Request simplified language for accessibility
- Specify number formatting (2 decimal places)
- Emphasize crop-specific strategies

## Error Handling

```python
import openai

def rec_gen_safe(factor, min, max, value, unit, crop):
    """Recommendation generator with error handling."""
    try:
        completion = client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[...]
        )
        return completion.choices[0].message.content
    except openai.RateLimitError:
        return "Service temporarily unavailable. Please try again."
    except openai.APIError as e:
        return f"API error: {str(e)}"
    except Exception as e:
        return "Unable to generate recommendation. Please contact support."
```
