## Overview

The harvest prediction model is the main ML component of MavunoX, designed to predict optimal harvest seasons by analyzing environmental factors across Nigeria, South Africa, Kenya, and Sudan. The model uses XGBoost classifier to predict when crops should be harvested based on soil pH, water availability, crop type, and country.

## Model Architecture

### Algorithm
- **Model Type**: XGBoost Classifier
- **Model Path**: `ML_Models/main_model_xgboost.pkl`
- **Framework**: XGBoost with scikit-learn integration

### Training Data

The dataset was sourced from the [Klusterthon Hackathon](https://www.kluster.africa/problem-statements/precision-farming-for-best-product-results-with-data) and includes:

- **Geographic Coverage**: Nigeria (majority), South Africa, Kenya, Sudan
- **Most Common Season**: Rainy season
- **Dominant Crop**: Maize
- **Key Features**: Temperature, humidity, pH, water availability, crop type, country

## Usage Example

```python
import pickle
import pandas as pd

# Load model and encoders
with open('ML_Models/main_model_xgboost.pkl', 'rb') as f:
    model = pickle.load(f)

with open('ML_Models/label_encoder.pkl', 'rb') as f:
    label_encoder = pickle.load(f)

with open('ML_Models/one_hot_encoder_info.pkl', 'rb') as f:
    one_hot_encoder_info = pickle.load(f)

# Prepare input data
new_entry = {
    'ph': 6.5,
    'water_availability': 180.5,
    'label': 'rice',
    'Country': 'Nigeria'
}

# Convert to DataFrame and encode
new_data = pd.DataFrame.from_dict({0: new_entry}, orient='index')
new_data_encoded = pd.get_dummies(new_data, columns=['Country', 'label'], dtype=int)

# Add engineered feature
new_data_encoded['ph_to_water_availability_ratio'] = (
    new_data_encoded['ph'] / new_data_encoded['water_availability']
)

# Align columns
new_data_encoded = new_data_encoded.reindex(
    columns=one_hot_encoder_info['columns'], 
    fill_value=0
)
new_data_encoded = new_data_encoded.drop('season', axis=1)

# Predict
predictions = model.predict(new_data_encoded)
harvest_season = label_encoder.inverse_transform(predictions)[0]

print(f"Predicted harvest season: {harvest_season}")
```

## Model Files

The harvest prediction system requires these pickle files:

- `main_model_xgboost.pkl` - Trained XGBoost classifier
- `label_encoder.pkl` - Season label encoder
- `one_hot_encoder_info.pkl` - One-hot encoding column information
- `crop_info.pkl` - Crop-specific optimal ranges for all factors

## Integration with Other Models

The harvest prediction model receives input from:

1. **Soil pH Predictor** - Converts RGB soil color to pH value
2. **Water Availability Model** - Converts RGB soil color to water availability score

See the respective model documentation for details on these preprocessing steps.
