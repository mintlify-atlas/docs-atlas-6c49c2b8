## Soil pH Model Evaluation

### Cross-Validation Approach

The soil pH predictor was evaluated using k-fold cross-validation to ensure robust performance estimates:

**Evaluation Metric: Root Mean Squared Error (RMSE)**

### Cross-Validation Results

**Average RMSE: 0.7049**

This value represents the average RMSE across all cross-validation folds, indicating the typical prediction error in pH units.

### RMSE Interpretation

Root Mean Squared Error measures the standard deviation of prediction errors:

```python
import numpy as np

# RMSE calculation
rmse = np.sqrt(np.mean((y_true - y_pred) ** 2))
```

An RMSE of 0.7049 means:
- Average prediction error is approximately ±0.7 pH units
- For soil pH typically ranging from 4-9, this represents ~10-15% relative error
- Predictions are sufficiently accurate for agricultural decision-making

### Cross-Validation Benefits

**Why K-Fold Cross-Validation?**

1. **Robust Estimates**: Averages performance across multiple train-test splits
2. **Efficient Data Use**: All data used for both training and validation
3. **Variance Assessment**: Reveals model stability across different data subsets
4. **Overfitting Detection**: Identifies if model generalizes well

**Typical Cross-Validation Process:**

```python
from sklearn.model_selection import cross_val_score
from xgboost import XGBRegressor
import numpy as np

# K-fold cross-validation
model = XGBRegressor()
scores = cross_val_score(
    model, 
    X, 
    y, 
    cv=5,  # 5-fold cross-validation
    scoring='neg_mean_squared_error'
)

# Calculate RMSE from negative MSE
rmse_scores = np.sqrt(-scores)
average_rmse = np.mean(rmse_scores)

print(f"Average RMSE: {average_rmse:.4f}")
```

## Evaluation Metrics Summary

### Classification Metrics (Main Model)

**Accuracy**: Overall correctness of predictions
```python
accuracy = (TP + TN) / (TP + TN + FP + FN)
```

**Precision**: Correctness of positive predictions
```python
precision = TP / (TP + FP)
```

**Recall**: Coverage of actual positives
```python
recall = TP / (TP + FN)
```

**F1-Score**: Harmonic mean of precision and recall
```python
f1_score = 2 * (precision * recall) / (precision + recall)
```

### Regression Metrics (Soil pH Model)

**Root Mean Squared Error (RMSE)**: Standard deviation of residuals
```python
rmse = sqrt(mean((y_true - y_pred)^2))
```

**Advantages of RMSE:**
- Same units as target variable (pH)
- Penalizes large errors more than small ones
- Widely used and interpretable
- Sensitive to outliers (encourages robust predictions)

## Recommendation Scoring

Beyond model predictions, MavunoX implements custom scoring for agricultural recommendations:

### Status Calculation

Binary indicator of whether a value falls within optimal range:

```python
def calc_status(value, value_range):
    if value >= value_range[0] and value <= value_range[1]:
        status = 1
    else:
        status = 0
    return status
```

### Scale Calculation

Gaussian-based similarity score showing how close a value is to optimal:

```python
def calc_scale(actual_number, input_number, k):
    absolute_difference = abs(actual_number - input_number)
    scale_value = 100 * math.exp(-k * absolute_difference**2)
    return scale_value
```

**Parameters:**
- `actual_number`: Measured value
- `input_number`: Optimal value for crop
- `k`: Sensitivity parameter (varies by factor)

**K Values by Factor:**
- pH: k = 0.6 (more sensitive to pH deviations)
- Water availability: k = 0.00025 (less sensitive to variations)
- Temperature: k = 0.04
- Humidity: k = 0.0015

The scale returns a value from 0-100, where 100 indicates perfect match with optimal conditions.
