## Introduction

MavunoX is a comprehensive machine learning system designed to predict harvest seasons for rural farmers in Africa. By analyzing environmental factors and soil characteristics, the system provides actionable predictions and recommendations to optimize crop yields.

## System Architecture

The MavunoX ML system employs a sophisticated three-model pipeline that works together to deliver accurate harvest season predictions:

## Core Components

### 1. Soil Profile Analysis

The system begins by processing soil images to extract two critical environmental parameters:

- **Soil pH**: Predicted using XGBoost regression on RGB color values and derived features (hue, saturation, brightness)
- **Water Availability**: Calculated using grayscale conversion and linear interpolation

### 2. Harvest Season Prediction

Using the extracted soil profile data along with crop type and country information, the main XGBoost classifier predicts the optimal harvest season with 89% accuracy.

### 3. AI-Powered Recommendations

The system generates custom farming recommendations using OpenAI's GPT-3.5 Turbo API, tailored to:
- Current environmental conditions
- Target crop requirements
- Optimal ranges for temperature, humidity, pH, and water availability

## Supported Crops

The system provides predictions and recommendations for 13 different crop types:

- **Cereals**: Maize, Rice
- **Pulses**: Chickpea, Kidneybeans, Lentil, Pigeonpeas, Mothbeans, Mungbean, Blackgram
- **Cash Crops**: Cotton, Jute
- **Vegetables**: Muskmelon, Watermelon

<Note>
Maize is the most frequently occurring crop in the training dataset, with the rainy season being the most common growing period.
</Note>

## Deployment

The system is deployed as a FastAPI application with Docker containerization, providing RESTful endpoints for:

- Soil profile analysis
- Harvest season prediction
- Environmental factor recommendations (temperature, humidity, pH, water availability)
