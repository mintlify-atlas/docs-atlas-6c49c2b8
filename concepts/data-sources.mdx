## Overview

MavunoX leverages multiple data sources to train its machine learning models and provide accurate predictions. This page details the datasets used, their origins, and the comprehensive crop information database.

## Primary Training Dataset

### Klusterthon Hackathon Dataset

The main prediction model is trained on agricultural data from the [Klusterthon Hackathon](https://www.kluster.africa/problem-statements/precision-farming-for-best-product-results-with-data).

**Dataset Characteristics:**

- **Geographic Coverage**: Nigeria, Kenya, South Africa, Sudan
- **Crops**: 13 different crop types
- **Environmental Factors**: Temperature, humidity, pH, water availability
- **Target Variable**: Harvest season (spring, summer, rainy, winter)
- **Size**: 1,400 records across different seasons and crops

## Soil pH Training Data

### RGB-to-pH Mapping Dataset

The Soil pH Predictor model is trained on data derived from academic research:

**Source**: Research paper "Determine the pH of Soil by Using Neural Network Based on Soil's Colour" ([ResearchGate](https://www.researchgate.net/publication/336681057_Determine_the_pH_of_Soil_by_Using_Neural_Network_Based_on_Soil's_Colour))

**Data Format**:
```
R, G, B, pH
133, 98, 30, 7.05
172, 139, 106, 6.81
176, 152, 114, 6.63
...
```

**Dataset Size**: 50 RGB-to-pH mappings covering pH range from approximately 5.5 to 8.4

<Note>
While the original research used neural networks, MavunoX implements an XGBoost regressor which achieved comparable accuracy with better interpretability.
</Note>

## Data Preprocessing

### Main Model Preprocessing Steps

1. **Feature Selection**
   - Dropped "harvest season" column (not needed for prediction)
   - Excluded "temperature" to prevent data leakage

2. **Feature Engineering**
   - Created `ph_to_water_availability_ratio` feature
   - This ratio captures the relationship between soil acidity and moisture

3. **Categorical Encoding**
   - One-hot encoded `Country` variable (4 categories)
   - One-hot encoded `label` (crop type) variable (13 categories)

4. **Target Encoding**
   - Label encoded `season` variable (4 classes)

### Train/Test Split

- **Training set**: ~80% of data
- **Test set**: ~20% of data (280 samples)
- Split maintains class distribution for balanced evaluation

## Data Quality Considerations

<Warning>
**Limitations to be aware of:**

1. **Geographic bias**: Nigeria is overrepresented in the dataset
2. **Seasonal imbalance**: Rainy season dominates the distribution
3. **Crop variety**: Limited to 13 crops (expansion planned)
4. **Water model simplicity**: Current grayscale-based approach is simplified
</Warning>

<Tip>
**Future enhancements planned:**

- Expand geographic coverage beyond current four countries
- Collect more balanced seasonal data
- Add support for additional crop varieties
- Implement more sophisticated water availability prediction using ML
- Incorporate satellite imagery and weather data
</Tip>

## Region-Specific Information

### Nigeria
- **Coverage**: Extensive across multiple agricultural zones
- **Primary crops**: Rice, maize, chickpea, cotton
- **Climate zones**: Tropical, sub-tropical

### Kenya
- **Coverage**: Major agricultural regions
- **Primary crops**: Maize, rice, beans
- **Climate zones**: Varied from arid to highland

### South Africa
- **Coverage**: Commercial farming areas
- **Primary crops**: Maize, cotton, various pulses
- **Climate zones**: Mediterranean to subtropical

### Sudan
- **Coverage**: Limited but growing
- **Primary crops**: Sorghum, millet, sesame (limited in current dataset)
- **Climate zones**: Arid to semi-arid
